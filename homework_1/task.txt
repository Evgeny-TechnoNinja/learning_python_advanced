ДЗ 1. Работа с bash из python

1. Ознакомиться с дополнительными материалами
2. Установить linux
3. Написать скрипт на python `dz1_1.py` который используя модуль subprocess будет (в указанном порядке):
    a. Выводить имя текущего пользователя
    b. Выводить путь к текущей директории
    c. Создавать папку `dz1` в текущей директории
    d. В папке `dz1` создавать файлы для каждого дня текущего месяца в формате `dd-mm-yyyy.log`
        (список файлов для создания должен создаваться с помощью генератора в python)
    e. менять владельца папки `dz1` и всех файлов в ней на пользователя `root`
    f. удалять 5 случайных файлов из папки `dz1`

Написать скрипт `dz1_2.py` который используя модуль subprocess будет (в указанном порядке):
    a. Копировать файл `dz1.py` в файл `dz1_run.py`
    b. Делать так чтобы он мог запуститься из bash
    c. Менять права доступа к нему на:
        i. Доступ полностью запрещен для всех кроме владельца
        ii. Владелиц может читать и запускать файл
    d. Запускать файл `dz1_run.py`
