Задание:

1.Реализовать класс Url который будет представлять URL
    a. Его параметрами должны быть все параметры URL (scheme, authority, path, query, fragment). Их должен принимать конструктор.
    b. Переопределить операцию == для Url. При сравнении Url со строкой должно возвращаться True в случае если строка соответствует Url.
    c.Реализовать класс HttpsUrl для которого scheme будет равна 'https' (Наследник Url)
    d. Реализовать класс HttpUrl для которого scheme будет равна 'http' (Наследник Url)
    e. Реализовать метод __str__ который будет формировать строку из Url
    f. Реализовать GoogleUrl и WikiUrl для которых authority будут соответственно 'google.com' и 'wikipedia.org'
2.Реализовать класс UrlCreator, с помощью которого можно будет создавать Url:
    a. При вызове метода _create должен возвращать Url
    b. Конструктор должен принимать параметры scheme и authority
    c. Методы __getattr__ и __call__ должны быть переопределены
    d. *args от __call__ должны добавлять этапы в лист path
    e. __getattr__ должен добавлять один этап в лист path
